# C++ 练习

个人 C++ 练习记录，每周发布学习总结。

## 工具

使用 VS Code 插件 Code Runner 编译，默认不支持输出不支持输入，改变配置项使其支持输入：

```json
{
    "code-runner.runInTerminal": false
}
```
